
# Biodata in webserver

Submission for pesilat 2023


## Run Locally

Clone the project

```bash
  git clone https://github.com/heavenlycreature/nodejs-biodata-pesilat2023.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run script
```
```bash
  go to your localhost:8080
```
